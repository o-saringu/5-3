プログラム復習コンテンツ

mouseX、mouseY変数

サンプルコードの結果

サンプルコード

　　 void setup(){

　　 　size(480, 120);

　　 　fill(0, 102);

　　 　noStroke();

　　 }

　　 void draw(){

　　 　ellipse(mouseX, mouseY, 9, 9);

　　 }

説明文

　mouseX、mouseY変数

　マウスによって座標を変える場合、mouseX変数、mouseY変数を用いる。

　draw()が実行されるたびに新しい円が1つ描かれる。　

　結果はマウスをあちこちに移動させ円の位置を変えた結果できたものである。

　fill関数により円の色を半透明に設定しているためマウスをゆっくり動かすと濃く見える。
